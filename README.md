# "# Protractor-Gradle" 

<i>Protractor</i> Mobile Web Automation Framework Data-Driven and, Component based using data sheet and, object repository.
</br></br>

# Execution:

* gradle wrapper
* gradle installGulp
* gradle clean build --refresh-dependencies
* gradle task e2e
* gradle wrapper installGulp clean build --refresh-dependencies task e2e --scan
* gradle --stop
</br></br>

# NPM Updates:

* sudo npm install npm-check --global --save
</br></br>
* sudo npm install npm-check-updates --global --save
* sudo npm install npm-check-updates --global --update-all --save
* sudo npm install
